# Verifier Agent

## Overview
The Verifier Agent serves as the deterministic validation engine for the generative pipeline. Acting as the mathematical "Discriminator" in a GAN-style system, it evaluates the coordinate layouts proposed by upstream Floor Plan Generators against the strict rulesets built by the Constraint Agent. It returns a proven SAT (Satisfied) or UNSAT (Unsatisfied) result alongside precise diagnostic feedback indicating exactly which constraints were violated.

## Key Features
- **Stateless Validation**: Operates entirely statelessly, evaluating the provided geometry and constraints instantly.
- **"No-Leak" Feedback**: Returns directional guidance and magnitude of errors (e.g., "Bedroom 1 is 10 sqft too small") without leaking exact coordinate corrections, forcing the Generator to iteratively "learn" the space.
- **Adapter Pattern Integration**: Encapsulates rigid external mathematical tools (like Z3 solvers) through a dynamic internal dispatcher.

## Input & Output Formats
- **Input**: A complex JSON payload containing:
  - `session_id`: Unique tracking ID.
  - `ruleset_schema`: The strict `exterior` and `interior` constraint dictionary generated by the Constraint Agent.
  - `layout_data`: The geometric proposal generated by the Floor Plan Generator (contains `boundary` and room `polygons`).
- **Output**: A JSON verification report containing:
  - `result`: Either `"SAT"` (Satisfied) or `"UNSAT"` (Unsatisfied).
  - `hard_violations`: An array of strings detailing exact rules broken (e.g., `"Bedroom 1 violates min_area"`).
  - `soft_violations`: Minor infractions or warnings.

## Directory Structure
- `api.py`: FastAPI server handling HTTP verification requests synchronously.
- `dispatcher.py`: Core routing and constraint logic processing engine.
- `contracts.py`: Pydantic Models defining the strict public API boundary.
- `z3_client.py`: Adapter for the external Z3 mathematical solver integration.
- `test_api.py`: Automated testing suite.
- `demo_inputs/`: Pre-configured JSON payloads for testing dynamic exterior and interior rules.
